# Frontend (Legacy Vite App)

This directory contains the **legacy Vite-based frontend shell** kept for compatibility and migration support.

## Canonical frontend

Active product development should happen in:

- `./ayn-landing-page` (Next.js + pnpm)

## Legacy app commands (this directory)

- Install: `npm install`
- Dev: `npm run dev`
- Lint: `npm run lint`
- Build: `npm run build`

## Notes

- Prefer the Next.js app for new UI/features.
- Keep this app only for migration-safe changes unless explicitly requested.
